---
description: Sincroniza github con todos los archivos.
---

1. Agrega todos los archivos (git add .) en un commit (git commit -m "Mensaje...") con mejores prácticas de escritura, de documentación dejando completamente claro qué fue lo que se desarrolló para que cualquier desarrollador pueda comprender lo que contiene este commit.
2. Haz un push (git push...) y sincroniza con github en la rama main.
