# Timeline Draw Syntax

## Syntax
```mermaid
timeline
    title History of Social Media Platform
    2002 : LinkedIn
    2004 : Facebook
         : Google
    2005 : Youtube
    2006 : Twitter
```

## Grouping
```mermaid
timeline
    title Timeline of Industrial Revolution
    section 17th-20th century
        1700 : Industry 1.0
        1800 : Industry 2.0
    section 21st century
        2000 : Industry 3.0
        2015 : Industry 4.0
```

## Styling
- `disableMultiColor`: Makes all periods same color
